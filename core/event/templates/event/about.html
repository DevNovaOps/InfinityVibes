PS 20: Event Management Portal - Web Development, AI

Problem Statement

There is currently no unified platform that enables users to seamlessly discover and connect with suitable event management vendors for diverse occasions such as birthdays, housewarmings, weddings, corporate events, exhibitions, or conferences. Users often struggle to identify the best vendors who can match their unique requirements just by sharing a similar picture from the internet or from a previous event.

In-Scope Features

  • Vendor registration with category, services, and portfolio details.
  • User search functionality using image-based input.
  • Recommendation engine to shortlist the top 10 vendors.
  • Event categories: personal (birthdays, weddings, housewarmings, casual gatherings) and professional (corporate events, conferences, exhibitions).
  • Vendor profiles with contact details, ratings, and previous work samples.

AI & Machine Learning Features

1. Image-Based Search with Computer Vision & Deep Learning

Users upload an image (e.g., decor inspiration, lighting setup, booth design). A pre-trained Vision-Language model (e.g., CLIP, BLIP, or DINOv2) converts the image into high-dimensional embeddings. A similarity algorithm (FAISS or ANN-based) matches the image against vendor portfolios. Results are ranked based on visual similarity, vendor quality metrics, and relevance to event type.

2. Vendor Portfolio Tagging Using AI

Vendor-uploaded images are auto-tagged using image classification and object detection models (YOLOv8, Detectron2). Tags like “floral décor”, “stage lighting”, and “open-air setup” help improve searchability and image understanding.

3. Generative AI for Style Translation

Users can optionally generate a visual style or mood board using text prompts (e.g., “Boho wedding at sunset with earthy tones”). GenAI models (Stable Diffusion, DALL·E) produce reference images to aid search or vendor communication.

4. Recommendation Engine with Hybrid Intelligence

Collaborative filtering and content-based filtering combine to provide personalized vendor recommendations. Vendors are ranked by image-style match, event category relevance, ratings, and performance history.

5. Natural Language Processing for Vendor Profiles & Queries

NLP models (e.g., BERT, RoBERTa) analyze vendor services, user feedback, and event descriptions to improve semantic matching and smart search capabilities.

6. Conversational AI for Planning Assistance (Future Scope)

A GenAI chatbot assistant could help users plan events, clarify needs, and navigate the platform with natural language queries.

Core Modules (MVP Scope)

  • Vendor Registration: AI-assisted onboarding, auto-tagging of portfolio images.
  • Image-Based Search: Upload image → extract features → retrieve & rank vendor portfolios.
  • Recommendation Engine: ML-based ranking of top 10 vendor matches.
  • Event Categories: Personal (birthday, wedding) and Professional (conferences, exhibitions).
  • Vendor Profiles: Ratings, past work, tags, services, and contact options.

Impact Metrics

  • Image-to-Vendor Matching Accuracy: Measures effectiveness of AI image search by % of test uploads returning relevant vendor portfolios.
  • Response Time for Image Search: Average time to return top 10 vendor matches after upload.
  • Relevance Score of Recommendations: User/judge feedback rating relevance on a 1–5 scale.
  • Tagging Precision: Manual validation of AI-generated portfolio tags.
  • Generative AI Prompt Success Rate: % of coherent and relevant images produced from text prompts.
  • Search-to-Click Ratio: User engagement measured by interaction with recommended vendors.
  • Repeat Query Capability: Ability to refine searches and get new results quickly.

Next Steps & Extensions (Optional)

  • Smart Vendor Chat Assistant leveraging LLMs (GPT-4, LLaMA) for natural language event planning.
  • Portfolio Quality Scoring Model using ML heuristics to rank vendor portfolio quality.
  • Feedback-Driven Learning Loop to personalize recommendations based on user likes/rejections.
  • Event Style Classifier using CNN or transfer learning for style-based filtering.
  • Interactive Moodboard Generator combining user images and GenAI outputs.
  • Location-Aware Search with map-based vendor filtering using mock or real geolocation data.
